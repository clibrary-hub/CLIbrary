# Omnibus skills

All-in-one skill pack for coding agents working with PostHog. Auto-synced daily from the PostHog [monorepo](https://github.com/PostHog/posthog) and [context-mill](https://github.com/PostHog/context-mill) — do not edit these skills directly, changes will be overwritten on the next sync.

Skills cover instrumentation (SDKs, feature flags, error tracking, LLM analytics, logs, product analytics), querying (HogQL examples, signals), and workflows (experiment analysis, flag auditing and cleanup, LLM cluster exploration).
