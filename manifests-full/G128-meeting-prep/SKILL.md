# meeting-prep Skill

## 功能概述

會議準備器。解決問題：開會前資料整理麻煩

## 輸入

- `run`: 執行主要功能

## 輸出

- JSON 結構化結果
- 終端美化輸出（rich）

## 使用範例

```bash
python meeting-prep.py run
```

## 技術棧

- Python 3.8+
- Click（CLI）
- Rich（終端美化）
- anthropic
