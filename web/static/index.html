Architecture

Agent intent │ ▼ Embed (multilingual-e5-base or Qwen3-Embedding-0.6B, ~10ms) │ ▼ Stage 1: FAISS cli_index → top-3 candidates (~5ms) │ ▼ Re-rank (MaxSim heuristic or Qwen3-Reranker cross-encoder) │ ├── low confidence + small gap → Clarify (return top-3 to LLM) │ └── high confidence │ ▼ Stage 2: example_index → best matching example │ ├── sim ≥ 0.85 → Path A: template fill (no LLM, ~80% of calls) │ └── sim < 0.85 → Path B: return params={} (caller fills via LLM)
